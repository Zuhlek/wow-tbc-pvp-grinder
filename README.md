A client-side TypeScript web tool for planning and tracking WoW Classic/TBC PvP honor grinding.

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Run tests
npm run test

# Run tests with coverage
npm run test:coverage

# Build for production
npm run build

# Lint
npm run lint
```
